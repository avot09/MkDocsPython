## 1. Что было сделано

### 1.1 Создана собственная тема
*   **Шаблон**: Создан файл `overrides/main.html` на Jinja2 с кастомными `<header>`, `<nav>`, `<footer>`.
*   **Стили**: Написан CSS в `src/custom.css` с переменными для цветов и шрифтов.
*   **Метаданные**: В шаблон добавлены теги `<meta>` (описание, автор, ключевые слова).

### 1.2 Настроен пайплайн сборки (CI/CD)
Создан файл `.github/workflows/deploy.yml`. При каждом пуше в `main` он:
1.  **Собирает CSS** через PostCSS (автопрефиксы + минификация).
2.  **Генерирует сайт** командой `mkdocs build`.
3.  **Минифицирует HTML** скриптом `scripts/minify.py`.
4.  **Проверяет HTML** на валидность утилитой `html5validator`.
5.  **Публикует сайт** на GitHub Pages.

### 1.3 Внедрены современные инструменты
*   **PostCSS**: Обработка стилей (`autoprefixer`, `cssnano`).
*   **Автоматизация**: Скрипт `npm run build:css` для сборки CSS.
*   **Валидация**: Обязательная проверка HTML перед деплоем.
